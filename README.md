# LiteAgent

LiteAgent is a compact, token-efficient tool-calling library for .NET that uses a text-first, TOON (Token-Oriented Object Notation) protocol to represent tool calls.

This repository contains the source package. Developer documentation is maintained in the `Docs/` folder.

- See the developer docs: [Docs/README.md](Docs/README.md)
- NuGet package: https://www.nuget.org/packages/LiteAgent/

If you are looking for examples, quick start, and the API reference, open `Docs/QuickStart.md`, `Docs/CoreConcepts.md`, and `Docs/API.md` respectively.
